## Getting Started

Live URL at: https://file-directory-tau.vercel.app/

Features:

- Toggle between a list view or a fold structure
- Re-fetch on returning to the tab
- Re-fetch every 30 seconds

Built with:

- NextJS
- Typescsript
- Tailwind
- Framer Motion for animations
